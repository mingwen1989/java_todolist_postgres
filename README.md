# Java To-do List w/ PostgreSQL

#### By _Ming Wen_

## Description

A basic to-do list written in Java and Spark that saves to a PostgreSQL database. Allows the user to create a new task and display a list of tasks, with persistent data.

## Setup and Installation Requirements

  Clone the repo.  
  Run a instance of the gradle web server with 'gradle run'.  
  Navigate to localhost:4567

  Additionally, tests can be run with 'gradle test'.
